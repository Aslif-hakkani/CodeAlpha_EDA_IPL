# 🏏 IPL Data Analysis — EDA

## Project Overview
Exploratory Data Analysis on IPL dataset (2008–2020)
Done as part of CodeAlpha Data Analytics Internship.

## Files
- `IPL_EDA.ipynb` — Main analysis notebook
- `matches.csv` — Match-level dataset

## Dataset
- `deliveries.csv` is not uploaded due to file size limit (27MB)
- Download it from Kaggle: https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020

## Tools Used
Python | Pandas | NumPy | Matplotlib | Seaborn

## Key Insights
- Mumbai Indians have the most wins in IPL history
- Virat Kohli is the highest run scorer
- Lasith Malinga is the highest wicket taker
- Teams prefer to field first after winning the toss
